# Acme Corp — Information Security Policy (Excerpt)

## Password requirements

All employees must use passwords that are at least 14 characters long
and combine uppercase letters, lowercase letters, digits and at least
one symbol. Passwords must be rotated every 90 days. Reuse of any of
the last 12 passwords is prohibited.

Multi-factor authentication is mandatory for every corporate account
that supports it, including email, source-control hosting, cloud
infrastructure consoles, and the internal VPN.

## Data classification

Acme classifies data into four tiers:

1. **Public** — material approved for external publication.
2. **Internal** — day-to-day operational data, not for external sharing.
3. **Confidential** — customer records, financial information, contracts.
4. **Restricted** — secrets, encryption keys, regulated personal data.

Restricted data must remain inside production environments and must
never be copied to laptops, USB drives, or personal accounts.

## Incident response

Suspected security incidents must be reported to security@acme.example
within one hour of discovery. The Security team will triage the report,
contain the issue, and coordinate communication with affected parties.

Employees must not attempt to remediate suspected breaches on their own
because uncoordinated action can destroy forensic evidence.
